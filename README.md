# README

## style

```
    layout: {
      align_type: center,
      border_width: 0,
      border_height: 0,
      candidate_spacing: 4,  # 候选项之间的间距
      corner_radius: 9,  # 面版圆角
      hilite_padding: 2,
      hilite_spacing: 5,
      line_spacing: 8,
      margin_x: 5,
      margin_y: 5,
      min_height: 0,
      min_width: 75,
      round_corner: 6,  # 候选项圆角
```

### layout

#### candidate_spacing

候选项与候选项之间的距离。

#### corner_radius

整个候选窗口的圆角，而非单个候选项的圆角。

#### round_corner

候选项的圆角。
